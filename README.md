## 🚀 How to Publish on GitHub Pages

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) and log in
2. Click the **"+"** icon → **"New repository"**
3. Name it: `portfolio` (or anything you like, e.g. `muhammad-umair-seo`)
4. Set it to **Public**
5. Click **"Create repository"**

### Step 2 — Upload the Files
1. On your new repo page, click **"Add file"** → **"Upload files"**
2. Drag and drop both files:
   - `index.html`
   - `README.md`
3. Scroll down and click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to your repo → click **"Settings"** tab
2. In the left sidebar, click **"Pages"**
3. Under **"Branch"**, select `main` and folder `/ (root)`
4. Click **"Save"**
5. Wait ~60 seconds, then your site will be live at:
   ```
   https://YOUR-USERNAME.github.io/REPO-NAME/
   ```

## ✏️ How to Customize

Open `index.html` in any text editor (Notepad, VS Code, etc.) and update:

| What to change | Where to find it |
|---|---|
| Your name & title | `<section class="hero">` |
| Email address | Contact section & footer |
| Project descriptions | `<section id="projects">` |
| Work experience | `<section id="experience">` |
| Skill percentages | `style="width:XX%"` on each bar |
| Accent color | `--green: #69bd27;` in `:root` |
